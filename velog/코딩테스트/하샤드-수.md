### 문제 설명
양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다. 자연수 x를 입력받아 x가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요.

### 풀이
숫자를 먼저 자릿수를 나눠 리스트화 하고, 리스트 안의 값들을 모두 더하자!
1. 숫자를 어떻게 리스트화?

#### list(map(int, n))
```
def solution(x):
    answer = list(map(int, x));
    return answer
```
**TypeError: 'int' object is not iterable**

=> 정수형은 반복자가 될 수 없으므로 문자로 처리해주어야 한다.

#### list(map(int, str(x)))
```
def solution(x):
    answer = list(map(int, str(x)));
    return answer
```

2. 리스트 안의 숫자들을 어떻게 더하기?
#### sum
리스트 안의 숫자들을 더하는 것은 sum 함수.

```
def solution(x):
    answer = list(map(int, str(x)));
    num = sum(answer);
    print(num);
```

3. 조건 분기해서 리턴해주자.
```
def solution(x):
    answer = list(map(int, str(x)));
    num = sum(answer);
    if x % num == 0 :
        return True;
    else:
        return False;
```

한 가지 주의할 점, 파이썬에서 리턴값이 될 T/F는 반드시 대문자로 시작해야한다.
