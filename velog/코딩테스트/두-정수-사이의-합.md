### 문제 설명
두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수, solution을 완성하세요.
예를 들어 a = 3, b = 5인 경우, 3 + 4 + 5 = 12이므로 12를 리턴합니다.

### 첫번째 풀이
a와 b 사이에 속한 모든 정수의 합? for range 를 이용해야겠다!
```
def solution(a, b):
    answer = 0
    if a > b :
        for i in range(b, a):
            answer += i;
        return answer;
    elif b > a:
        for i in range(a, b):
            answer += i;
        return answer;
    else :
        return a;
```

결과는 실패.
사이의 모든 값을 더한 결과값이 리턴되지 않고, 단순히 a와 b를 더한 값만이 리턴되고 있었다.

### 마지막 풀이
원하는 결과값을 도출해내지 못하는 이유는 범위 설정이 잘못되었기 때문이다.



```
def solution(a, b):
    answer = 0
    if a > b :
        for i in range(b, a+1):
            answer += i;
        return answer;
    elif b > a:
        for i in range(a, b+1):
            answer += i;
        return answer;
    else :
        return a;

```
a가 b보다 작은 경우, a부터 b 사이에 속한 모든 정수의 합을 리턴하고자 했지만,
b에 1을 더해주지 않았기 때문에 사실상 b는 더해지고 있지 않았던 것이다.
index 설정에 주의하자......  **항상 기본에 충실하기 !**

### 추가: Javascript로도 풀어본다면?
```
function solution(a, b) {
  let answer = 0;
  if (a > b) {
    for (let i = b; i <= a; i++) {
      answer += i;
    }
    return answer;
  } else if (b > a) {
    for (let i = a; i <= b; i++) {
      answer += i;
    }
    return answer;
  } else {
    return a;
  }
}

```
