### 문제 설명
함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

### 풀이
```
def solution(n):
    return list(map(int, str(n)))
```
리스트로 만들어서 정렬을 해야겠다...!
정렬을 어떻게 하지?

#### list.sort() 와 내장함수 sorted()
리스트의 메소드인 sort()는 리스트 자체를 변경한다.
sorted()는 기존의 리스트를 변경하는 것이 아니라 정렬된 새로운 리스트를 반환한다.
```
def solution(n):
    a = sorted(list(map(int, str(n))))
    return a
```

역순으로 반환하려면 reverse=True를 넣어줘야한다.
```
def solution(n):
    a = sorted(list(map(int, str(n))), reverse=True)
    return a
```


### 리스트를 문자열로 변환하기
#### 1. 첫번째 시도: join

```
def solution(n):
    arr = sorted(list(map(int, str(n))), reverse=True);
    result = "".join(arr);
    return result
```

-> **TypeError: sequence item 0: expected str instance, int found**

int형을 join하려고 보니 생기는 에러라고 한다.

```
def solution(n):
    arr = sorted(list(map(int, str(n))), reverse=True);
    result = "".join((map(str, arr)));
    return int(result)
```

성공! 그런데 map을 이렇게 여러 차례 사용하는 게 맞는 걸까...?

리팩토링...
```
def solution(n):
    return int("".join(sorted(str(n), reverse=True)))
```


#### 2. 두번째 시도: for in 반복문
```
def solution(n):
    arr = sorted(list(map(int, str(n))), reverse=True);
    
    str = '';
    
    for s in arr:
        str += s
        
    return str 
```
-> **UnboundLocalError: local variable 'str' referenced before assignment**
str이라는 변수가 할당되기 전에 참조되었다고 한다.
내장 함수 str과 충돌이 일어나는 것으로 보인다. 예약어는 변수명으로 사용하면 안된다 작성자가 바보인 듯

```
def solution(n):
    arr = sorted(list(map(int, str(n))), reverse=True)
    
    result_str = ''
    
    for s in arr:
        result_str += str(s)
        
    return int(result_str) 

```

성공!
