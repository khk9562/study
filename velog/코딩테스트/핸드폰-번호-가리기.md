# 문제 설명
프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 *으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

### 제한 조건
phone_number는 길이 4 이상, 20이하인 문자열입니다.

<br/>

## 문제풀이 - Python3

먼저 파이썬 슬라이싱 기법이 리스트 말고 문자에도 적용이 되는지 확인했다.
테스트 결과, 성공! 그래서 매우 쉽게 한번에 결과를 낼 수 있었다.
```
def solution(phone_number):
    ends = phone_number[-4:]
    length = len(phone_number) - 4
    result = "*" * length + ends
    return result
```

폰넘버의 뒤 4자리만 따로 빼서 ends로 정의하고,
폰넘버의 길이를 측정하여 4를 빼주어 추출했다.
그 길이만큼 문자 별을 결과에 더해주고, 나머지 뒤에 ends도 더해주면 정답!

다른 분들은 한줄로 코딩했을 거 같아 풀이를 확인해보니 역시...
```
return "*"*(len(s)-4)+s[-4:]
```
의미는 동일하지만,
변수 선언없이 바로 리턴하는게 더 효율적인가?라는 고민을 하게 되었다.

<br />

## 문제풀이 - Javascript
문제는 자바스크립트... 요 몇 주 C언어, 자바, 파이썬만 하다보니 벌써 가물가물해진다..
파이썬 슬라이싱 기능을 대체할 함수가 자바스크립트에는 없는 걸로 알아서
for문을 활용하기로 생각.

먼저 파라미터로 주어지는 스트링을 리스트화 하기 위해
스프레드 연산자를 사용하여 list라는 변수에 할당하였다.
```
    let list = [...phone_number]
```

그리고 list의 길이를 추출하여 list안의 아이템의 인덱스가 list의 길이에서 4를 뺀 수보다 작으면 "*"를, 크면 list[i]를 미리 선언한 result 변수에 추가하려고 하였다.

```
function solution(phone_number) {
    let list = [...phone_number]
    let result;
    console.log(list)
    for (i=0; i<list.length; i++) {
        console.log("i",i, list.length - 4)
        if(i < list.length - 4) {
            result += "*"
        } else {
            result += list[i];
        }
    } 
    return result;
}
```
**
여기서 오류 발생!**

	실행한 결괏값 "undefined*******4444"이 기댓값 "*******4444"과 다릅니다.
    
저 맨앞의 undefined 값만 사라지면 정답과 동일한 결괏값이 출력되는데,
어째서 undefined가 첫번째로 출력되는 걸까? 아마 인덱스 값 설정이 문제겠지..?

모르겠어서 gpt한테 물어보니,,,,,

> 문제의 원인은 변수 result를 초기화하지 않고 사용했기 때문입니다.


아! 빈문자열로 result 값을 정의하지 않고 그대로 더한채로 사용해서
정의하지 않은 result값의 undefined가 함께 출력되는 거였다....

결과적으로 내가 작성한 정답은 아래와 같다.
```
function solution(phone_number) {
    let list = [...phone_number]
    let result = "";
    for (i=0; i < list.length; i++) {
        if(i < list.length - 4) {
            result += "*"
        } else {
            result += list[i];
        }
    } 
    return result;
}
```

**다른 사람의 풀이를 참고하니, 파이썬의 슬라이싱이 자바스크립트에 없다고 단정했는데... 생각해보니 자바스크립트에도 slice함수가 있다...... 자괴감이 느껴진다..**

```
function hide_numbers(s){
    var result = "*".repeat(s.length - 4) + s.slice(-4);
    return result;
  }
```
