## 문제 설명
어떤 정수들이 있습니다. 이 정수들의 절댓값을 차례대로 담은 정수 배열 absolutes와 이 정수들의 부호를 차례대로 담은 불리언 배열 signs가 매개변수로 주어집니다. 실제 정수들의 합을 구하여 return 하도록 solution 함수를 완성해주세요.

### 첫번째 풀이(Python3)
각 배열의 index를 이용해 for in range로 모든 요소를 더해야겠다!
```
def solution(absolutes, signs):
    print(signs);
    sum = 0;
    for i in range(1, len(absolutes) + 1):
        if signs[i] == True:
            sum += absolutes[i];
        else:
            sum -= absolutes[i];
        return sum;
```

실행한 결과값이 기댓값과 다르다고 출력된다. 문제가 뭘까?

#### 1. 변수명으로 예약어 사용하지 말 것.
충돌 발생 가능성을 고려하여 sum 대신 result로 변수명을 수정하였다.
#### 2. range 범위 수정
배열 인덱스는 0부터 시작하므로 1부터 len(absolutes) + 1이 아니라, len(absolutes)로 설정하여야 한다.
#### 3. return 위치 수정
기존 코드에서 return 문은 for 루프 내에 위치해서 첫번째 반복에서 함수가 종료되므로 for 반복문이 완료된 외부로 이동시켜야 한다.

### 정답 풀이(Python3)
```
def solution(absolutes, signs):
    result = 0
    for i in range(len(absolutes)):
        if signs[i] == True:
            result += absolutes[i]
        else:
            result -= absolutes[i]
    return result

```


### 풀이(Javascript)
```
function solution(absolutes, signs) {
    result = 0;
    for (i=0; i < absolutes.length; i++) {
        if (signs[i] == true) {
            result += absolutes[i];
        } else {
            result -= absolutes[i];
        }
    }
    return result;
}
```
