# 문제
도현이는 바구니를 총 N개 가지고 있고, 각각의 바구니에는 1번부터 N번까지 번호가 순서대로 적혀져 있다. 바구니는 일렬로 놓여져 있고, 가장 왼쪽 바구니를 1번째 바구니, 그 다음 바구니를 2번째 바구니, ..., 가장 오른쪽 바구니를 N번째 바구니라고 부른다. 

도현이는 앞으로 M번 바구니의 순서를 역순으로 만들려고 한다. 도현이는 한 번 순서를 역순으로 바꿀 때, 순서를 역순으로 만들 범위를 정하고, 그 범위에 들어있는 바구니의 순서를 역순으로 만든다.

바구니의 순서를 어떻게 바꿀지 주어졌을 때, M번 바구니의 순서를 역순으로 만든 다음, 바구니에 적혀있는 번호를 가장 왼쪽 바구니부터 출력하는 프로그램을 작성하시오.

### 입력
첫째 줄에 N (1 ≤ N ≤ 100)과 M (1 ≤ M ≤ 100)이 주어진다.

둘째 줄부터 M개의 줄에는 바구니의 순서를 역순으로 만드는 방법이 주어진다. 방법은 i j로 나타내고, 왼쪽으로부터 i번째 바구니부터 j번째 바구니의 순서를 역순으로 만든다는 뜻이다. (1 ≤ i ≤ j ≤ N)

도현이는 입력으로 주어진 순서대로 바구니의 순서를 바꾼다.

### 출력
모든 순서를 바꾼 다음에, 가장 왼쪽에 있는 바구니부터 바구니에 적혀있는 순서를 공백으로 구분해 출력한다.

<br />

# 풀이

### 1차 시도

1부터 N까지의 배열을 만들고, 입력받은 범위(i~j)만큼만 역순으로 만들기 위해
arr[i:j]만 역순으로 정렬 시키고 전 후로 붙여넣는 방법으로 시도했다.

```
N, M = map(int, input().split())
arr = []
for i in range(N): arr.append(i+1)
# print(arr)
for i in range(M):
    i,j = map(int, input().split())
    arr = arr[:i-1] + arr[i:j].sort(reverse=True) + arr[j:]
print(arr)
```

결과는 역시나 오류.

    arr = arr[:i-1] + arr[i:j].sort(reverse=True) + arr[j:]
_**TypeError: can only concatenate list (not "NoneType") to list**_


### 2차 시도
그렇다면 새로운 배열을 만들어서 거기에 append 시킨다면?!


```
N, M = map(int, input().split())
arr = []
result = []
for i in range(N): arr.append(i+1)
# print(arr)
for i in range(M):
    i,j = map(int, input().split())
    result.append(arr[:i-1])
    result.append(arr[i:j].sort(reverse=True))
    result.append(arr[j:])
print(result)
```

출력값은......ㅋㅋ...
[[], None, [3, 4, 5], [1, 2], None, [5], [], None, [5], [1], None, [3, 4, 5]]

**일단 
1. 왜 None이 들어가있으며, 
2. append한 값들이 배열형태로 들어가는지
기초부터 익혀야겠다....**

> 
**1. sort 함수는 None을 반환하는 성질을 가진다.**
arr[i:j].sort(reverse=True)는 arr[i:j]의 사본을 내림차순으로 정렬하려고 시도하지만, sort() 메서드는 리스트 자체를 정렬하고 None을 반환한다.
따라서, reverse 함수 또는 [::-1]슬라이싱 메소드를 사용할 것!

**2. 왜 배열로 들어가는가?**
배열에 arr[i]와 같이 인덱스 값으로 아이템을 뽑아내면 배열 아닌 아이템만 들어가지만, slicing으로 해서 삽입하면 배열형태로 들어간다..... 당연한 걸 간과했다


### 문제 풀이 방향 수정
result 라는 새로운 배열을 선언할 게 아니라
기존 배열에서 i-1부터 j에 한해서만 역순으로 값 변경해주기!

```
N, M = map(int, input().split())
arr = [i+1 for i in range(N)]
for i in range(M):
    i,j = map(int, input().split())
    arr[i-1:j] = arr[i-1:j][::-1]
```

공백으로 구분해 출력해야하므로 join 사용해주면 성공!

```
N, M = map(int, input().split())
arr = [i+1 for i in range(N)]
for i in range(M):
    i,j = map(int, input().split())
    arr[i-1:j] = arr[i-1:j][::-1]
    
print(" ".join(str(i) for i in arr))
```
