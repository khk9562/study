#### 문제 설명

임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다.
n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.


#### 제한 사항

- n은 1이상, 50000000000000 이하인 양의 정수입니다.



### 첫번째 풀이

```
def solution(n):
    x = 0
    if x % 1 == 0:
        return (x + 1) ** 2;
    return -1
```

x를 어떻게 n의 제곱근으로 표현할 수 있는지 모르겠다.
검색의 힘을 빌려,,,


### 두번째 풀이

```
def solution(n):
    x = n ** (1/2)
    if x % 1 == 0:
        return (x + 1) ** 2;
    return -1
```

#### **n의 1/2 제곱은 n의 제곱근이다.**


### 또 다른 풀이

제곱근을 반환하는 함수 sqrt를 사용하기 위해 math모듈을 import 한다.

```
import math

def solution(n):
    x = math.sqrt(n)
    if x != int(x):
        return -1
    return math.pow(x+1, 2)
```

math.pow(x, n)은
x를 n제곱한 결과값을 리턴한다.

결론적으로, 파이썬 기초 문법에 대한 지식이 부족하다. 공부하자.
