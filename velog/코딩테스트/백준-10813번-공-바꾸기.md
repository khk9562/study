# 문제
도현이는 바구니를 총 N개 가지고 있고, 각각의 바구니에는 1번부터 N번까지 번호가 매겨져 있다. 바구니에는 공이 1개씩 들어있고, 처음에는 바구니에 적혀있는 번호와 같은 번호가 적힌 공이 들어있다.

도현이는 앞으로 M번 공을 바꾸려고 한다. 도현이는 공을 바꿀 바구니 2개를 선택하고, 두 바구니에 들어있는 공을 서로 교환한다.

공을 어떻게 바꿀지가 주어졌을 때, M번 공을 바꾼 이후에 각 바구니에 어떤 공이 들어있는지 구하는 프로그램을 작성하시오.




### 입력
첫째 줄에 N (1 ≤ N ≤ 100)과 M (1 ≤ M ≤ 100)이 주어진다.

둘째 줄부터 M개의 줄에 걸쳐서 공을 교환할 방법이 주어진다. 각 방법은 두 정수 i j로 이루어져 있으며, i번 바구니와 j번 바구니에 들어있는 공을 교환한다는 뜻이다. (1 ≤ i ≤ j ≤ N)

도현이는 입력으로 주어진 순서대로 공을 교환한다.


### 출력
1번 바구니부터 N번 바구니에 들어있는 공의 번호를 공백으로 구분해 출력한다.

<br/>

## 요약 
 1. N개의 바구니, 각각의 바구니에 1부터 N번까지 번호가 매겨져있음
 2. 바구니에는 공이 1개씩 들어있고, 처음에는 바구니에 적혀있는 번호와 같은 번호가 적힌 공이 들어있음
 3. 앞으로 M번 공을 바꿀 계획
 4. 공을 바꿀 바구니 2개를 선택하고, 두 바구니에 들어있는 공을 서로 바꿈
 5. 공을 어떻게 바꿀지가 주어졌을 때, M번 공을 바꾼 이후 각 바구니에 어떤 공이 들어있는지 구하기
 
 
 <br />
 
# 풀이
 

 
 첫째 줄에 N과 M을 입력받은 후
 1부터 N까지의 1차원 배열을 만들고,
 M개의 순서, 즉 i와 j를 입력받았다.
 그리고 arr[i-1]와 arr[j-1]의 순서를 바꿔주었다.
 
 
 ```
N, M = map(int, input().split())
arr = []
for index in range(N):
    arr.append(index+1)
for index in range(M):
    i,j = map(int, input().split())
    arr[i-1], arr[j-1] = arr[j-1], arr[i-1]
print(arr)
```
 
 
 
그런데 출력을 배열 그대로 하면 파이썬에서는 [a,b,c,d,e]의 형태로 그대로 출력되므로
**배열을 문자열화** 해주어야한다.
 
그래서 arr.join("")이라고 해주었다가
자바스크립트 문법과 혼동했음을 깨달아
"".join(arr)로 수정해주었는데,

 
     print(" ".join(arr))
    
_**TypeError: sequence item 0: expected str instance, int found**_

타입에러가 났다. int를 str화 하는 과정에서 에러가 발생한 것이다.
그러면 배열에서 하나씩 아이템을 가져오며 str화 해주어야 하는데, 어떻게 해야하는지 모르겠어서.... 구글 검색의 도움을 받았다...!

```
**print(" ".join(str(a) for a in arr))**
```
