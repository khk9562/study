## 문제 설명
String형 배열 seoul의 element중 "Kim"의 위치 x를 찾아, "김서방은 x에 있다"는 String을 반환하는 함수, solution을 완성하세요. seoul에 "Kim"은 오직 한 번만 나타나며 잘못된 값이 입력되는 경우는 없습니다.

## 문제 풀이 - Javascript
```
function solution(seoul) {
    let x = 0;
    while(seoul[x] != "Kim") {
        x ++;
    }
     return `김서방은 ${x}에 있다`;
    
}
```
한 번에 성공!

## 문제 풀이 - Python3
### 첫번째 시도
```
def solution(seoul):
    answer = 0;
    for x in seoul:
        answer += 1;
        if x == "Kim":
            return `김서방은 {answer}에 있다`;

```

_**SyntaxError: invalid syntax**_

return한 내용이 형식에 맞지 않다는 오류가 나타났다.
자바스크립트와 달리 백틱으로 리턴하는 문법이 아닌 것으로 추정된다.
구글 검색을 통해 알아보니,
백틱이 아니라 따옴표를 사용하며 그 앞에 f를 붙여 표현해야 원하는 기능 수행이 가능하다고 한다.
그리고 이것을 파이썬에서는 formatting이라고 칭한다.
> 변수를 넣고자 하는 문장 맨 앞에 f를 붙이고 변수에 {}를 씌운다.


### 두번째 시도
```
def solution(seoul):
    answer = 0;
    for x in seoul:
        answer += 1;
        if x == "Kim":
            return f"김서방은 {answer}에 있다";
```

_**실행한 결괏값 "김서방은 2에 있다"이 기댓값 "김서방은 1에 있다"과 다릅니다.**_

이번엔 오류는 나타나진 않았지만, 실행 결과가 기대와 달랐다.
이는 answer에 +1을 하는 위치가 잘못된 것으로 생각된다.


### 세번째 시도 & 정답

```
def solution(seoul):
    answer = 0;
    for x in seoul:
        if x == "Kim":
            return f"김서방은 {answer}에 있다";
        answer += 1;
```
두번째 시도가 잘못된 이유는 "Kim"을 찾기 전에 +1을 하기 때문에 리턴할 때 index가 적절하지 않다 !!!
