## 문제 설명
array의 각 element 중 divisor로 나누어 떨어지는 값을 오름차순으로 정렬한 배열을 반환하는 함수, solution을 작성해주세요.
divisor로 나누어 떨어지는 element가 하나도 없다면 배열에 -1을 담아 반환하세요.

### **제한사항**
arr은 자연수를 담은 배열입니다.
정수 i, j에 대해 i ≠ j 이면 arr[i] ≠ arr[j] 입니다.
divisor는 자연수입니다.
array는 길이 1 이상인 배열입니다.

<br />

## 문제 풀이 - Javascript
### 첫번째 풀이
```
function solution(arr, divisor) {
    let result= [];
    for (x of arr){
        if (x % divisor == 0) {
            result.push(x);
        }
    }
    return result.sort();
}
```
실패!

_**실행한 결괏값 [10,5]이 기댓값 [5,10]과 다릅니다.**_

sort() 기본값이 오름차순인데 이렇게 출력된 게 이해가 안돼서 검색해보았다.

> 많은 개발자 분들이 sort() 함수를 사용하실 때 중요한 부분을 간과하시다가 당혹스러운 경험을 하시게 되는데요. 바로 정렬하기 전에 배열 내의 값을 내부적으로 문자열로 변환한다는 사실입니다.

그렇다. 나도 당황스러웠다!
하지만 이제는 절대 까먹지 않을 것 같다.
그렇다면 숫자 기준으로 제대로 오름차순 나열을 하려면 어떻게 해야할까?

### sort함수로 숫자 배열 정렬
```
arr.sort((a, b) => a - b) // 오름차순 정렬
arr.sort((a, b) => b - a) // 내림차순 정렬

```

이 검색 결과를 토대로 내 코드에 적용시켰다.
<br/>

### 두번째 풀이
```
function solution(arr, divisor) {
    let result= [];
    for (x of arr){
        if (x % divisor == 0) {
            result.push(x);
        }
    }
    return result.length > 0 ? result.sort((a, b) => a - b) : [-1];
}
```
성공!

하지만 리팩토링이 필요할 거 같다.
<br/>

### 마지막 풀이
```
function solution(arr, divisor) {
    const result = arr.filter(x => x % divisor === 0).sort((a, b) => a - b);
    return result.length > 0 ? result : [-1];
}

```

## 문제 풀이 - Python3
### 첫번째 풀이
```
def solution(arr, divisor):
    result = [];
    for x in arr:
        if x % divisor == 0 :
            result.append(x).sort();
    if len(result) == 0 :
        return [-1];
    else:
        return result;
    
```

_**AttributeError: 'NoneType' object has no attribute 'sort'**_
첫번째 시도 실패!
append 하는 값이 없으면 이러한 에러가 뜨니까 값이 있는 경우에만 sort해줘야 한다.

<br />
### 두번째 풀이
```
def solution(arr, divisor):
    result = [];
    for x in arr:
        if x % divisor == 0 :
            result.append(x);
    if len(result) == 0 :
        return [-1];
    else:
        return result.sort();
```

두번째 시도 실패...!
return 하면서 sort하니 null을 반환한다.
sort() 함수는 원본 리스트를 정렬하고 반환하는 값은 없기 때문에 return에 바로 작성하면 안된다..
이렇게 return에 바로 작성해서 의도한대로 반환하려면 sorted(result)를 사용해야한다.
아니면 for문이 끝나고 result를 먼저 sort시켜줘야 한다.

### 세번째 풀이
```
def solution(arr, divisor):
    result = [];
    for x in arr:
        if x % divisor == 0 :
            result.append(x);
    if len(result) == 0 :
        return [-1];
    else:
        return sorted(result);
```
**또는**
```
def solution(arr, divisor):
    result = [];
    for x in arr:
        if x % divisor == 0 :
            result.append(x);
    result.sort();
    if len(result) == 0 :
        return [-1];
    else:
        return result;
```
_**성공!**_

다른 사람들의 코드도 잠깐 봤는데
```
def solution(arr, divisor): return sorted([n for n in arr if n%divisor == 0]) or [-1]
```

이런 함수가 있었다. or이라는 것도 처음 봤다...!
