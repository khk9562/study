### VPN이란?


**Virtual Private Network**의 약자로, 번역하면 가상 사설망이라는 뜻.


→ 사설망(Private Network)이라는건 공용 네트워크(Public Network)가 아닌 외부에 공개되지 않은 네트워크를 말하는 것.

> 원격지와 목적지 서버 사이에 가상의 터널을 만들어 멀리 떨어진 원격지에서도 마치 랜선으로 연결된 것처럼 사설망을 이용할 수 있게 하는 것

*(이미지 생략)*


### 사용 이유

1. 편리성
원격지에서도 사설망에 접속하여 회사 내에서 일하는 것 같은 환경을 만들 수 있음
2. 암호화
해킹으로부터 안전함, 외부에서 누군가 데이터를 훔쳐봐도 내용을 알 수 없음
3. 기밀성
인터넷을 이용하는 내 정보를 숨길 수 있음. 인터넷 이력, 접속 사이트 등
4. IP 우회
해외 VPN 서버를 통해 인터넷에 접속함으로써 IP 국가 우회가 가능
→ 국내 접속 불가 사이트 접속 가능
5. 고정 IP 부족 문제 해결
VPN IP를 통해 고정 IP를 사용하는 듯한 환경 구축이 가능
